## À propos

Ce portfolio est un site personnel créé pour présenter mes projets, ma méthodologie de travail et mes compétences en développement web.

Développé dans le cadre de ma formation à la **HEPL - Techniques Infographiques option Web**, ce projet a été pensé 
autour d'un thème détective avec des tableaux d'enquête.

---

## Objectifs

- Présenter mes projets et mes compétences
- Démontrer ma méthodologie de travail
- Respecter les critères académiques (responsive, accessibilité, SEO)
- Créer une expérience utilisateur fluide et intuitive

---

## Technologies

| Technologie | Utilité |
|-------------|---------|
| HTML5 | Structuration sémantique du contenu |
| CSS3 + SASS | Mise en forme modulaire et animations |
| JavaScript | Interactivité et effets dynamiques |
| PHP | Traitement du formulaire de contact |
| WordPress | Gestion dynamique du contenu |
| Fancybox | Galeries d'images interactives |

---

## Structure du site

- **Accueil** : Présentation générale et appel à l'action
- **Mes Projets** : Showcase des réalisations avec descriptions détaillées
- **À Propos** : Présentation personnelle et compétences
- **Me Contacter** : Formulaire de contact fonctionnel
- **Système multilingue** : Contenu disponible en FR et EN

---

## Fonctionnalités

- Site dynamique avec WordPress pour une gestion facile du contenu
- Formulaire de contact avec validation et gestion des erreurs
- Galeries d'images interactives avec Fancybox
- Système de thème clair/sombre (lampe interactive)
- Indicateur de scroll pour montrer la progression de lecture
- Menu burger responsive pour mobile
- Animations fluides au survol et au focus
- Design responsive sur tous les appareils

## Installation

### Prérequis

- Serveur avec PHP 7.4+
- WordPress 5.0+

### Étapes

1. Cloner le repository
   ```bash
   git clone https://github.com/Camara-Mohamed/wordpress-portfolio.git
   cd wordpress-portfolio
   ```

2. Installer les dépendances frontend
   ```bash
   npm install
   ```

3. Compiler les fichiers SASS (ouvrir 2 terminaux)

   Terminal 1 - Compilation automatique
   ```bash
   npm run watch
   ```

   Terminal 2 - Serveur de développement
   ```bash
   npm run serve
   ```
